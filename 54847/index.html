运动微分方程的建立

根据牛顿第二运动定律，质点的加速度和受到的力成正比，二作用力一般来说可以是位矢 r\boldsymbol{r}r 速度 r˙\boldsymbol{\dot r}r˙ 以及时间 t\boldsymbol{t}t 的函数。
质点运动的微分方程可以写为：

mr¨=F(r,r˙,t)m \boldsymbol{\ddot r} = \boldsymbol{F}(\boldsymbol{r}, \boldsymbol{\dot r}, \boldsymbol{t} ) mr¨=F(r,r˙,t)

直角坐标系下的三个标量方程为：

mx¨=Fx(x,y,z;x˙,y˙,z˙;t)my¨=Fy(x,y,z;x˙,y˙,z˙;t)mz¨=Fx(x,y,z;x˙,y˙,z˙;t)\begin{aligned} &m \ddot{x}=F_x(x, y, z ; \dot{x}, \dot{y}, \dot{z} ; t) \\ &m \ddot{y}=F_y(x, y, z ; \dot{x}, \dot{y}, \dot{z} ; t) \\ &m \ddot{z}=F_x(x, y, z ; \dot{x}, \dot{y}, \dot{z} ; t) \end{aligned} ​mx¨=Fx​(x,y,z;x˙,y˙​,z˙;t)my¨​=Fy​(x,y,z;x˙,y˙​,z˙;t)mz¨=Fx​(x,y,z;x˙,y˙​,z˙;t)​

平面极坐标系下的质点微分运动方程为：

m(r¨−rθ˙2)=Fr(r,θ;r˙,θ˙;t)m(rθ¨+2r˙θ˙)=Fθ(r,θ;r˙,θ˙;t)\begin{gathered} m\left(\ddot{r}-r\dot{\theta}^2\right)=F_r(r, \theta ; \dot{r}, \dot{\theta} ; t) \\ m(r \ddot{\theta}+2 \dot{r} \dot{\theta})=F_\theta(r, \theta ; \dot{r}, \dot{\theta} ; t) \end{gathered} m(r¨−rθ˙2)=Fr​(r,θ;r˙,θ˙;t)m(rθ¨+2r˙θ˙)=Fθ​(r,θ;r˙,θ˙;t)​

如果质点受到某种约束,例如被限制在某曲线或曲面上运动,不能脱离该线或该面而作任意的运动并占据空间任意的位置,则叫非自由质点.此时该线或该面叫约束,而该线或该面的方程则叫做约束方程
令 F\boldsymbol{F}F 代表主动力，R\boldsymbol{R}R 代表约束反作用力，则质点的运动微分方程为：

mr¨=F(r,r˙,t)+Rm\boldsymbol{\ddot r} = \boldsymbol{F}(\boldsymbol{r},\boldsymbol{\dot r}, t)+ \boldsymbol{R} mr¨=F(r,r˙,t)+R
