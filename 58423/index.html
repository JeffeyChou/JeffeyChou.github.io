二元关系

  • 序偶(ordered pair)：由两个元素按照一定的次序组成的二元组称为有序偶对，简称序偶。
  • 笛卡尔积(Cartesian product)：设 A1,A2,⋯ ,AnA_1, A_2, \cdots, A_nA1​,A2​,⋯,An​ 是 nnn 个集合, 称集合

A1×A2×⋯×An={(a1,a2,⋯ ,an)∣ai∈Ai∧i∈{1,2,3,⋯ ,n}}A_1 \times A_2 \times \cdots \times A_n=\left\{(a_1, a_2, \cdots, a_n)| a_i \in A_i \wedge i \in\{1,2,3, \cdots, n\}\right\} A1​×A2​×⋯×An​={(a1​,a2​,⋯,an​)∣ai​∈Ai​∧i∈{1,2,3,⋯,n}}

为集合 A1,A2,⋯ ,AnA_1, A_2, \cdots, A_nA1​,A2​,⋯,An​ 的笛卡尔积。

  • 二元关系(relation)： 设 A,BA, BA,B 为两个非空集合, 称 A×BA \times BA×B 的任意子集 RRR 为从 AAA 到 BBB 的一个二元关 系,简称关系(relation). 如果 A=BA=BA=B, 则称 RRR 为 AAA 上的一个二元关系.

cij={1,∃k, 使得 aik=1 且 bkj=10, 否则 c_{i j}= \begin{cases}1, & \exists k, \text { 使得 } a_{i k}=1 \text { 且 } b_{k j}=1 \\ 0, & \text { 否则 }\end{cases} cij​={1,0,​∃k, 使得 aik​=1 且 bkj​=1 否则 ​

  • 复合关系：设 A,B,CA, B, CA,B,C 是三个非空集合, RRR 是从 AAA 到 BBB 的关系, SSS 是从 BBB 到 CCC 的关系 (也 可简单地描述为 R:A→B,S:B→CR: A \rightarrow B, S: B \rightarrow CR:A→B,S:B→C ), 则 RRR 与 SSS 的复合关系 (合成关系) (composite relation) R∘SR \circ SR∘S 是从 AAA 到 CCC 的关系,并且

R∘S={(x,z)∣x∈A∧z∈C∧(∃y)(y∈B∧xRy∧ySz)}R \circ S=\{(x, z)\mid x \in A \wedge z \in C \wedge(\exists y)(y \in B \wedge x R y \wedge y S z)\} R∘S={(x,z)∣x∈A∧z∈C∧(∃y)(y∈B∧xRy∧ySz)}

运算“∘\circ∘”称为复合运算(composite operation).

  • 递归关系：设 RRR 是集合 AAA 上的关系。 RRR 的 nnn 次幂 Rn(n=1,2,3,⋯ )R^n(n=1,2,3, \cdots)Rn(n=1,2,3,⋯) 递归地定义为

R1=R 和 Rn+1=Rn∘RR^1=R \text { 和 } R^{n+1}=R^n \circ R R1=R 和 Rn+1=Rn∘R

  • 逆关系：设 A,BA, BA,B 是两个非空集合, RRR 是从 AAA 到 BBB 的关系, 则从 BBB 到 AAA 的关系

R−1={(b,a)∣<a,b>∈R}R^{-1}=\{(b, a)\mid<a, b>\in R\} R−1={(b,a)∣<a,b>∈R}

称为 RRR 的逆关系 (inverse relation), 运算“-1”称为逆运算 (inverse operation).

  • 自反关系：若对每个元素 a∈Aa \in Aa∈A 有 (a,a)∈R(a, a) \in R(a,a)∈R 那么定义在集合 AAA 上的关系 RRR 称为自反的。即集合中的每个元素都与自身有关系。
  • 对称与反对称：对于任意 a,b∈Aa, b \in Aa,b∈A, 若只要 (a,b)∈R(a, b) \in R(a,b)∈R 就有 (b,a)∈R(b, a) \in R(b,a)∈R, 则称定义在集合 AAA 上的 关系 RRR 为对称的。对于任意 a,b∈Aa, b \in Aa,b∈A, 若 (a,b)∈R(a, b) \in R(a,b)∈R 且 (b,a)∈R(b, a) \in R(b,a)∈R, 一定有 a=ba=ba=b, 则称定义 在集合 A\mathrm{A}A 上的关系 RRR 为反对称的。
  • 传递性：若对于任意 a,b,c∈A,(a,b)∈Ra, b, c \in A,(a, b) \in Ra,b,c∈A,(a,b)∈R 并且 (b,c)∈R(b, c) \in R(b,c)∈R 则 (a,c)∈R(a, c) \in R(a,c)∈R, 那么定义在 集合 AAA 上的关系 RRR 称为传递的。

集合 AAA 上的关系是传递的当且仅当对 n=1,2,3,⋯n=1, 2, 3, \cdotsn=1,2,3,⋯ 有 Rn⊆RR^n \subseteq RRn⊆R 。
